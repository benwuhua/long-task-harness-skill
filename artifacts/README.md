# Artifacts

`artifacts/` is the default location for durable engineering evidence produced during development, testing, and diagnosis.

Suggested subdirectories:

```text
artifacts/
  logs/
  screenshots/
  traces/
  reports/
  reproductions/
```

Use this directory for logs, screenshots, trace files, test reports, and captured command output that supports a feature status change, bug report, or recovery decision.

Use [`docs/templates/artifacts-readme-template.md`](../docs/templates/artifacts-readme-template.md) as the starter template for project-specific conventions.
